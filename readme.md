# AQP

Yet Another Rewrite of QP - this time with AngularJS

## Build
	git clone
	npm update
	bower update
	sudo npm start


## Mongodb Notes
* [http://docs.mongodb.org/manual/tutorial/install-mongodb-on-ubuntu/]
* sudo apt-get install mongodb-org
* sudo mkdir /data/db
* sudo chmod 0755 /data/db
* mongod -f ./mongod.conf
